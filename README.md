NELSNMP
=======

Nelsnmp is a wrapper around pysnmp to make it easier to use SNMP against network equipment.

It is used my [Nelmon](https://networklore.com/nelmon/) and the [Cisco SNMP Ansible modules](https://networklore.com/ansible-cisco-snmp/)
